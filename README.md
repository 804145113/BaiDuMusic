# BaiDuMusic
#
百度随心听网页端解析的数据，具体的抓包过程就不啰嗦了，还是要费点小时间。 
用swift写的一个简单的播放器，可在线播放音乐电台，支持后台播放，封面信息，可显示歌手名，专辑名，当前播放时间等，项目用到了第三方开源框架，有OC的，也有纯swift的，可以当做学习swift的例子。
